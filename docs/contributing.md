## Contribuer au projet

Merci pour votre intérêt ! Voici comment contribuer :

1. Forker le dépôt et créer une branche de fonctionnalité :

   git checkout -b feat/ma-fonctionnalite

2. Respecter le style du projet et ajouter des tests pour toute nouvelle fonctionnalité.
3. Ouvrir une Pull Request décrivant le changement.

Guidelines :
- Écrire des commits clairs et atomiques.
- Mettre à jour la documentation (dans `docs/`) si nécessaire.
- Vérifier que le projet build et que les tests passent localement.

Contact :
- Ouvrez une issue si vous trouvez un bug ou une amélioration.
