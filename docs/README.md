## Documentation du projet final_tp

Ce dossier contient la documentation complète du projet "API_Jeu-main" (frontend admin-dashboard + backend-API).

Structure du dossier de documentation :

- `architecture.md` : vue d'ensemble de l'architecture, schéma, composants.
- `setup.md` : installation locale, dépendances, commandes.
- `frontend.md` : détails sur le frontend `admin-dashboard`, variables d'environnement, build.
- `backend.md` : détails sur le backend `backend-API`, routes, modèles, démarrage.
- `api.md` : spécification des endpoints disponibles (routes, payloads, réponses).
- `deployment.md` : guide de déploiement (Render, conseils, scripts).
- `env_variables.md` : liste des variables d'environnement et exemples (`.env.example`).
- `troubleshooting.md` : erreurs courantes et comment les résoudre (inclut l'erreur `ERR_CONNECTION_REFUSED` sur localhost).
- `contributing.md` : guide pour contribuer au projet.

Utilisez ces documents pour configurer, développer, tester et déployer l'application.
